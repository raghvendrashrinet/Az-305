#### Cosmos APIs
* SQL API ⭐⭐⭐
* MongoDB API
* Cassandra API
* Gremlin API
* Table API

Cosmos API :refers to the communication layer or the "language engine" you use to talk to the database.,it stores data in a universal backend format, but it allows developers to talk to it using the specific database languages they already know.
