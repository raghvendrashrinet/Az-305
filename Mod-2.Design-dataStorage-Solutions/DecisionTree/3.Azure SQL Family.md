## Azure SQL Family Decision Tree  

```
    Need SQL

        │
        ▼
 Single Database?

      │
  Azure SQL Database

  Many databases?

      │
  Elastic Pool

 Need Instance-level Features?

      │
 Managed Instance

 Need Full Windows Access?

      │
 SQL on Azure VM

```

| Service            | Best For             |
| ------------------ | -------------------- |
| Azure SQL Database | Cloud-native apps    |
| Elastic Pool       | Many small databases |
| Managed Instance   | SQL migrations       |
| SQL VM             | Full SQL control     |
