## Managed Identity Decision Tree
```
Azure Resource?

      │
      ▼
Need Azure API?

      │
      ▼
Yes

      │
      ▼
Use Managed Identity

Need Shared Identity?

      │
    Yes
      │
User Assigned MI

One Resource Only?

      │
      ▼
System Assigned MI
```

* One VM --> 	`System Assigned MI`
* Shared across many resources -->	`User Assigned MI`
