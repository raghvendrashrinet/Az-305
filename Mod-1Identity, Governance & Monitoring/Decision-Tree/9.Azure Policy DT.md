## Azure Policy Decision Tree

```
Need to enforce standards?

          │
          ▼
Stop deployment?
      │
     YES
      │
Deny Policy

Need automatic correction?
      │
     YES
      │
DeployIfNotExists

Need audit only?
      │
     YES
      │
Audit Policy

Need enforce tags?
      │
     YES
      │
Modify Policy
```

| Requirement          | Effect            |
| -------------------- | ----------------- |
| Block deployment     | Deny ⭐⭐⭐      |
| Report only          | Audit             |
| Auto-create settings | DeployIfNotExists |
| Add missing tags     | Modify            |
