## Log Analytics Decision Tree

```
Need log search?

        │
        ▼
Multiple Azure resources?

        │
        ▼
Log Analytics Workspace

Need KQL Queries?

        │
        ▼
Log Analytics

Need Sentinel?

        │
        ▼
Use same Workspace
```
Log Flow
```
Azure Resources
      │
Diagnostic Settings
      │
Log Analytics Workspace
      │
KQL Queries
      │
Alerts / Workbooks
```
